## Engineering focus

- **Applied AI:** LLM API, RAG, AI agents, prompt contracts, structured outputs, evaluation, repair и quality gates.
- **Backend:** FastAPI, Node.js, PostgreSQL, Redis, workers, очереди, webhooks, OAuth и payment systems.
- **Reliability:** идемпотентность, retries, leases, дедупликация, checkpoints и восстановление после сбоев.
- **Frontend and product:** TypeScript, React, Next.js, UX, onboarding, conversion flows и административные интерфейсы.
- **Quality:** UI/API/DB testing, Playwright, Postman, Selenium, нагрузочное тестирование, анализ логов и release validation.
- **Delivery:** Docker, CI/CD, Railway, Selectel, Caddy, App Store Connect и GitHub Actions.
